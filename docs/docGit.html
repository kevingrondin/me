Git

Dans notre dossier du projet tapez

git init

Pour faire un commit sur tout le projet avec un message.

git commit -a -m "Mon message"

remplace les etapes suivante

# Mettre a jour un fichier
git add "monfichier.txt"
# Tout mettre à jour
git add .

# Enregistrer les mise à jour avec message
git commit -m "Message pour le commit"

Pour voir les traces de tout modification

git log

Si on veux se position à différent moment du projet

# Revenir à un moment du projet
git checkout SHADuCommit

# Venir sur la branche principale
git checkout master

Différente methode pour annuler un commit

# Creer l'inverse d'un commit
git revery SHADuCommit

# Modifier le message d'un commit
git commit --amend -m "Nouveau message"

# Annuler tout les modif si on a pas encore fait de commit
git reset --hard

GITHUB

Recuperer un projet github

git clone lienGithub

publier sur github

git push origin nmaster

recupérer en local les possible mise a jour

git pull origin master

Les branches

Visualiser les branches d'un projet une * apparait pour voir sur quel branche vous êtes

git branch

#Creer une branche
git branch nouvelle-branche

#Se placer dans une branche
git checkout nouvelle-branche

#(Concaténer les deux étape précedente) Creer une branche et s'y positioner directement
git checkout -b ma-branche

Si on veux recuperer les mise a jour effectuer dans brancheB vres brancheA

git checkout brancheA
git merge brancheB

.gitignore

Lister les dossiers et fichier dans un fichier .gitignore

Pull request

Faire un fork du projet depuis le site github

# Cloner le en local sur votre machine
git clone monliengithub

# creer une branche
git checkout -b new-feature

# faite vos modif et commiter
git commit -m "Modification terminé"

#Publier
git push origin new-feature

Placez vous sur votre fork Github, sur votre nouvelle branche, et cliquez sur "Compare & pull request"
